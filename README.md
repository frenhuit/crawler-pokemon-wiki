# Crawler for Pokemon wiki

# Description

Simply get data from Pokemon wiki(https://bulbapedia.bulbagarden.net/wiki/Main_Page), then insert them into MongoDB and Elasticsearch.

# Getting Started

## Dependencies

* Make sure to install dependencies listed in requirements.txt.

## Installing

* Replace parameters in ./spiders/settings.py with your local parameters (such as MangoDB) 
* Make sure MangoDB and ElasticSearch are installed.

## Executing program

* Run crawler.py
